= highByte()


// ÜBERSICHTSABSCHNITT STARTET
[#overview]
--

[float]
=== Beschreibung
Liest das most-significant (linkeste) Byte eines Wortes (oder das zweitkleinste Byte eines größeren Datentypes).
[%hardbreaks]


[float]
=== Syntax
`highByte(x)`


[float]
=== Parameter
`x`: Wert von beliebigen Typ.

[float]
=== Rückgabewert
Das geforderte Byte. Datentyp: `byte`.
